# chibi_reporter

Generates Reports for Chibi

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'chibi_reporter', :git => git://github.com/dwilkie/chibi_reporter.git
```

And then execute:

```shell
$ bundle
```
